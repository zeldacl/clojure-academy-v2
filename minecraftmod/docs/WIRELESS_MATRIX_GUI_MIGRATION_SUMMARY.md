## 核心特性

### 1. 动态UI切换

**3种状态**：
1. **未初始化 + 所有者** → 显示初始化表单（SSID输入框 + 密码输入框 + INIT按钮）
2. **未初始化 + 非所有者** → 显示提示消息（"Network not initialized"）
3. **已初始化** → 显示网络信息（SSID + 密码，所有者可编辑）

**实现技术**：
- `atom`存储动态容器
- `swap!`原子更新
- 函数式重建

### 2. 网络管理

**功能**：
- 创建网络（初始化）
- 修改SSID
- 修改密码
- 查询网络信息（SSID、密码、负载）

**权限**：
- 只有所有者可以初始化和修改
- 客户端UI隐藏 + 服务端验证

### 3. 特殊布局

**插槽**：
- 3个Constraint Plate（三角形布局）
- 1个Matrix Core（中心位置）

**坐标**：
```
       Plate1 (60,0)
          /  \
         /    \
        /      \
   Plate2      Plate3
   (35,49)     (86,49)
   
     Core (60,25)
      [中心]
```

- 网络消息处理：~180行
- 总计：~320行

**复用率**：55.7%

### 2. XML驱动

**优势**：
- 布局与逻辑分离
- 修改布局无需重新编译
- 可视化设计（未来可支持可视化编辑器）

**示例**：
```xml
<Property name="prop_ssid">
  <label>SSID</label>
  <editable>true</editable>
  <requiresOwner>true</requiresOwner>
</Property>
```

### 3. 函数式编程

**无副作用组件创建**：
```clojure
(defn create-property-widget [spec tile player on-change]
  (comp/property-field
    :label (get-in spec [:content :label])
    :editable (and (get-in spec [:content :editable])
                   (is-owner? tile player))
    :on-change on-change))
```

**动态重建**：
```clojure
(defn rebuild-info-panel! [container-atom ...]
  (swap! container-atom assoc :children [])
  (let [new-panel (build-panel ...)]
    (swap! container-atom update :children conj new-panel)))
```
